FakeNews Detector v0.1

AI-assisted fact-checker for Chrome, Edge, and Firefox — verify news, videos, and social media using your own LLM API key.

What it does

FakeNews Detector analyzes the text, audio, and video content of web pages you choose to examine, sends it to the AI provider you configure (using your own API key), and returns a per-claim verdict with confidence scores and source links — directly in a side panel.

This is a decision-support aid, not an infallible lie detector. Always verify critical claims independently.

Key features

  • Text, tab audio, and microphone analysis
  • 14 built-in providers (Anthropic, OpenAI, Gemini, DeepSeek, Qwen, Kimi, GLM, MiniMax, Grok, Mistral, Groq, OpenRouter, Ollama, Custom)
  • Silent mode with Telegram alerts and email via webhook (n8n / Zapier / Make)
  • Session conclusion and history views
  • Real-time VU meter for audio capture
  • Day / night theme
  • No relay server — your browser talks directly to the AI provider you choose
